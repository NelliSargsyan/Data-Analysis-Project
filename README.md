This project loads, preprocesses, queries clinic-related data using Python's libraries Pandas and Numpy along with PostgreSQL as a query tool, and Tableau for viusalizations. The pipeline.py file loads the data which was initially fully clean. Then, missing and invalid values are injected into the data to be able to clean, validate and check the data quality later. After the validation step, we connect to PostgreSQL to query the data (in order to connect to Postgres please enter your own password in line 163 of pipeline.py). Through the SQL query we generate insights about the dataset, at the same time, connecting the three datasets to each other on clinic_id. That connection makes it easier to create interesting and meaningful visualizations in Tableau. The viusalization results are also attached!
